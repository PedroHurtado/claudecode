## Sistema visual (ambos decks)
- Fuente: Plus Jakarta Sans (DS). Mono del sistema para prompts/código.
- Fondos: blanco (contenido) + rojo `#DF2F3A` (portada, divisores, principios). Máx 2.
- Tarjetas "terminal" oscuras (`#111827`) para prompts/código — guiño a Claude Code.
- Patrón contenido: overline rojo en mayúsculas + título extrabold + cuerpo.
- Type scale 1920×1080: display 88 / title 60 / subtitle 40 / lead 36 / body 30 / small 24 / mono 24.
